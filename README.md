# Windows Server Active Directory Homelab

## Overview
This project documents the creation of a Windows Server 2022 homelab environment using VirtualBox to develop practical IT infrastructure and systems administration skills.

## Technologies Used
- VirtualBox
- Windows Server 2022
- Active Directory
- DNS
- Windows 11

## Objectives
- Install Windows Server
- Configure Active Directory
- Promote server to Domain Controller
- Create users and groups
- Join Windows client to domain
- Practise troubleshooting and administration

## Skills Developed
- Windows Server administration
- Active Directory management
- Virtualisation
- Networking fundamentals
- Troubleshooting

-Initial VM setup and Windows Server installation completed prior to documentation stage.
